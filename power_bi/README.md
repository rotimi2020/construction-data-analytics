# Power BI Dashboard

This folder holds the Power BI dashboard for the Construction Operations Analytics project.

The dashboard turns the construction site data into interactive visuals – so you can see costs, materials, equipment usage, and site progress at a glance.

---

## Files

| File | Description |
|------|-------------|
| `Operations_Performance_Dashboard.pbix` | The main Power BI report. |
| `dashboard.pdf` | A static PDF export – no Power BI needed. |

---

## 📂 Folders

### `dax/`
Documentation for all the DAX logic used in the report:

| File | Description |
|------|-------------|
| `measures.md` | All the DAX measures (Total Cost, Running Total, Cement Reconciliation, etc.). |
| `calculated_columns.md` | All the calculated columns (Month Name, Cost Category, etc.). |

### `screenshots/`
Quick previews of the dashboards:

| File | Description |
|------|-------------|
| `cost_summary.PNG` | Cost breakdown by location and activity. |
| `site_operations.PNG` | Site activity and progress tracking. |
| `store_records.PNG` | Material inventory and cement reconciliation view. |

---

## Dashboard Features

- KPI cards for quick project health checks
- Cost breakdowns by activity and location
- Equipment usage tracking
- Material inventory and cement reconciliation
- Interactive filters to explore the data your way

---

## Tools Used

- Power BI Desktop
- Power Query (data prep)
- DAX (calculations and measures)

---

## What This Shows

This dashboard proves I can take cleaned project data and build a business-facing report that's actually useful – interactive, clear, and focused on the numbers that matter for tracking a construction project.

---

## How to Use

1. Open `Operations_Performance_Dashboard.pbix` in Power BI Desktop (it's free).
2. Play with the filters and slicers to explore the data.
3. If you don't have Power BI, open `dashboard.pdf` or the screenshots for a quick preview.
