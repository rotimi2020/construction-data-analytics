# Excel Reports

This folder contains the main Excel workbook for the Construction Operations Analytics project, plus a set of PDF reports exported for quick viewing.

The workbook takes real construction site records and turns them into structured data, summary reports, KPI dashboards, and business insights.

---

## Files

| File | Description |
|------|-------------|
| `Operations_Analytics_Portfolio.xlsx` | The full Excel workbook – documentation, cleaned data, summaries, dashboards, and analysis. |
| `pdf/` | PDF exports of the key reports for quick viewing without Excel. |

---

## Workbook Structure

The workbook has 11 worksheets, designed to be read in order:

| Worksheet | What's inside |
|-----------|---------------|
| `00_COVER` | Project cover and navigation. |
| `00_EXECUTIVE_SUMMARY` | Key findings and project performance at a glance. |
| `01_OVERVIEW` | Project background, scope, and methodology. |
| `02_DATA_DICTIONARY` | Definitions for every field in the dataset. |
| `03_MASTER_LOG` | The cleaned construction site records. |
| `04_ACTIVITY_SUMMARY` | What work was done, when, and how much. |
| `05_MATERIAL_SUMMARY` | Material deliveries, usage, and cement reconciliation. |
| `06_COST_SUMMARY` | Breakdown of costs by activity, labour, equipment, and more. |
| `07_KPI_DASHBOARD` | Interactive dashboard showing key metrics. |
| `08_MATERIAL_COST_ANALYSIS` | Material consumption trends and cost analysis. |
| `09_CHALLENGES_AND_RESOLUTIONS` | Problems faced during the project and how they were solved. |
| `10_PIVOT_ANALYSIS` | Pivot tables for deeper data exploration. |

## How to Use

1. Open `Operations_Analytics_Portfolio.xlsx` in Excel (2016 or later).
2. Browse the worksheets from `00_COVER` to `10_PIVOT_ANALYSIS` to follow the full workflow.
3. For a quicker overview, open any PDF in the `pdf/` folder.
